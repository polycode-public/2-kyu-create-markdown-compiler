Markdown Compiler Sample Document

This document demonstrates the core features supported in Chunk 1: headings, paragraphs, and inline formatting.

Inline Formatting

You can use bold text, italic text, inline code, and strikethrough to format content.

Mixed Formatting

You can combine formats: bold italic, bold with code, and italic with strikethrough.

XSS Safety

User content is safely escaped: <script>alert('xss')</script> becomes harmless text.

The compiler supports all standard heading levels from h1 to h6.
